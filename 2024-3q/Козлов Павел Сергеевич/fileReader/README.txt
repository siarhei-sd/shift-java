Использовал Maven 3.9.0 и Java 11 (можно выше)
Из сторонних библиотек использовал парсер - commons-cli (ссылка в помнике есть)
Для сборки исполняемого архива -jar использовал maven-assembly-plugin. Он сразу пакует все нужные библиотеки в архив. Танцев с бубном не нужно. Для корректной сборки желательно в коммандной строке прописать: mvn clean compile assembly:single
Текстовые файлы утиллита читает из той же папки, где лежит.
