FileContentFilter 1.0
Утилита для сортировки и сбора статистики содержимого 
текстовых файлов.

В проекте использовались:
-Java 17.0.12
-фреймворк для автоматизации сборки проектов Maven 3.8.7
-библиотека Apache common-cli 1.9.0
https://commons.apache.org/proper/commons-cli/

Инструкция по сборке проекта.

Для сборки проекта используя командную строку необходимо:
-распаковать архив в нужную директорию
-перейти в корневую папку проекта 
FileContentFilter/
-выполнить команду mvn clean install
После завершения сборки перейдите в директорию 
FileContentFilter/target 
в которой будет находиться файл FileContentFilter-1.0.jar.
Запустите его командой java -jar FileContentFilter-1.0.jar.

Также можно воспользоваться стандартными средствами сборки
jar-файла IntelliJ Idea.
