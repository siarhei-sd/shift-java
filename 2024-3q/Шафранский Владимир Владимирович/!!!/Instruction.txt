Версия JAVA 22
Сборка и запуск проекта - в консоли операционной системы.
Использованы стандартные пакеты JAVA

Исходный файл MultiFileSplitter.java

Скомпилиовать:
1. Запустить консоль на ПК
2. Указать путь к папке, где расположен Java файл (в Windows команда cd /d d:/!!!/ , где cd - смена папки, /d указание смены диска и далее путь к папке, где лежит скрипт)
3. Запускаем компиляцию командой javac MultiFileSplitter.java
4. Запуск класса MultiFileSplitter с параметрами -о (адрес папки, где будут находиться результаты работы скрипта), -p (Префикс для названия файлов), -a (указание на дополнение файла данными
java MultiFileSplitter -o d:/!!!/output -p result_ -a "in1.txt" "in2.txt"
	В результате будет создано три файла с префиксом одинаковым для всех файлов в папке output
	
Для вывода статистики использовать команды
java MultiFileSplitter -o D:/!!!/output -p result_ -f "in1.txt" "in2.txt"
java MultiFileSplitter -o D:/!!!/output -p result_ -s "in1.txt" "in2.txt"
