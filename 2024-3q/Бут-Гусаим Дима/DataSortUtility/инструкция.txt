Версия: java 22.0.2
Система сборки: Oracle OpenJDK 22.0.2

В архиве находятся:
1) 3 файла с кодом: Main.java, 
                    CommandLineOptions.java, 
                    DataFilter.java.
2) 2 тестовых файла: in1.txt,
                     in2.txt.
3) jar-файл для запуска: DataSortUtility.jar

Для запуска в командной строке необходимо:
1) перейти в нужную директорию
2) ввести команду java -jar DataSortUtility.jar [опции и исходные файлы]

Все опции, реализованные в данной утилите, соответсвуют условию задания.
