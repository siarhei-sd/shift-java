FILE UTIL

Java version: 1.8
Build system: Maven 4.0.0


Инструкция запуска:

1. Открываем консоль
2. cd \directory\path\fileUtil
3. mvn package (должен быть установлен maven, нужен интернет) (достаточно выполнить единожды)
4. cd target
5. java -jar file-util.jar <нужные опции> <files.txt> (файлы для фильтрации нужно предварительно создать)

Примечание:
Проект должен был работать с Maven-wrapper`ом, чтобы не требовалась установка Maven`a, однако из-за политики google (нельзя отправить исполняемые файлы по почте) не удалось это сделать.
