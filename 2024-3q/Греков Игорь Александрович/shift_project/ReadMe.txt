Версия Java 8 и выше
Maven - 4.0.0

Инструкция:
1. Распакуйте архив с исходным кодом в удобное место.

2. Откройте терминал (MacOS) или командную строку (Windows) и перейдите в корневую директорию проекта.
   
   MacOS: Используйте команду cd, чтобы перейти в нужную директорию:

      cd путь/до/директории/проекта

   Windows: Используйте команду cd, чтобы перейти в нужную директорию:

      cd путь\до\директории\проекта

3. Выполните сборку проекта с помощью Maven:
   mvn clean package

   эта команда скомпилирует проект и создаст исполняемый JAR файл в папке target.

4. Находясь в корневой директории проекта, запустите проект командой (для MacOS используйте "/" вместо "\"):
   java -jar target\Shift_proj-1.0-SNAPSHOT.jar "путь к файлам от корневой директории"

   общий вид команды:
   java -jar "путь до исполняемного файла"\Shift_proj-1.0-SNAPSHOT.jar "опции" "путь к файлам" 

5. Опции:

   '-o "путь"' - сохранение отфильтрованных файлов по указанному пути. По умолчанию сохраняется в папке выполнения команды.
   '-p "префикс"' - добавляет префикс к названию файла вида "префикс_название" - "префикс"integers.txt. Названия по умолчанию "integers", "floats", "strings".
   '-s'/'-f' - предоставляет короткую ('-s') или полную ('-f') статистику. По умолчанию предоставляется короткая статистика ('-s').
   '-a' - дополнение информацией уже существующих файлов. По умолчанию файлы перезаписываются.

Пример запуска с опциями:

   MacOS: java -jar target/Shift_proj-1.0-SNAPSHOT.jar -o путь/вывода/файлов -p result -f -a путь/до/входных/файлов

   Windows: java -jar target\Shift_proj-1.0-SNAPSHOT.jar -o путь\вывода\файлов -p result -f -a путь\до\входных\файлов
